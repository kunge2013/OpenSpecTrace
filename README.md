# OpenSpec 闭环开发增强套件  基于 OpenSpec 框架的二次开发，为规范驱动开发注入知识管理能力。  ## 核心能力  📦 代码逻辑归档（/opsx:code-anysic）   每次 /opsx:apply 后自动分析变更代码，按功能模块将业务逻辑归档至 business-logic/ 目录，   沉淀为结构化的业务知识库。  🔍 业务逻辑检索（/opsx:business-search）   通过关键词自然语言检索已归档的业务规则、功能逻辑和代码入口，   快速定位历史实现，辅助新功能开发与问题排查。  ## 闭环工作流  Propose → Apply → 逻辑归档 → Archive → 业务检索 → 循环迭代
